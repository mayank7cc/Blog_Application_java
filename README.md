This is a full stack blogging web application developed using the Spring Boot framework. 
The application allows users to create, edit, view, and delete blog posts with details such as title, content, and author. 
It uses MySQL for data persistence and Thymeleaf for rendering a responsive and premium frontend UI.
The project demonstrates complete CRUD functionality with proper backend integration and follows standard Spring Boot application architecture.

Features:
Create new blog posts with title, content, and author,
Edit existing blog posts,
Delete blog posts,
View all blogs and individual blog details,
Full stack CRUD implementation,
Clean UI rendered using Thymeleaf,
MySQL database integration.

Technologies Used:
Java,
Spring Boot,
Spring Data JPA,
MySQL,
Thymeleaf,
HTML and CSS.

Requirements:
Java JDK 8 or higher,
MySQL Server,
Maven.

Build and run the application:
mvn spring-boot:run,
Open your browser and navigate to:
http://localhost:8080.
